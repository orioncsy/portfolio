서비스 개요

시험 문제를 출제하고 응시할 수 있는 웹 서비스를 개발하였습니다. 주관식, 객관식 문제를 출제할 수 있고 답안을 작성하여 시험 문제를 등록하면 출제한 문제를 응시하고 채점하여 결과를 알려주는 서비스입니다.


핵심 개발 사항

  • 시험 문제 출제 페이지 구현
  • 주관식 페이지와 객관식 페이지 구현
  • 객관식에 보기 추가 기능 구현
  • 시험 문제 응시 페이지 구현
  • 정답 표시와 오답 표시 구현 및 총 점수 표시
